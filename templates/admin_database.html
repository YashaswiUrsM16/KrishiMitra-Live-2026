Technical Architecture Brief

1. Storage Architecture

Data is persisted using **SQLite 3**, a self-contained, high-reliability database engine.

  • Engine: SQLite 3 (Portable & Fast)
  • Mapper: SQLAlchemy ORM
  • Type: Relational Schema (SQL)

2. Data Security

Protecting vulnerable farmer data through industry-standard encryption protocols.

  • Encryption: One-way Bcrypt Hashing.
  • Injections: 100% SQL-Injection Protected.
  • Sessions: Secure Cryptographic Cookies.

3. Data Integrity

A multi-module schema tracking every aspect of the crop lifecycle.

  • AI Insights: Historical Crop Predictions.
  • Finances: Income/Expense auditing.
  • Audit: Real-time Activity Tracking.
