## 📖 About the Project

**KrishiMitraAI** is an intelligent web-based farming assistant that helps Indian farmers make better agricultural decisions using Agentic AI and Machine Learning.

The system provides:
- 🌱 **AI-powered crop recommendations** based on soil NPK values, pH, temperature, humidity and rainfall.
- 🐛 **Plant disease detection** using vision analysis.
- 🤖 **Agentic AI Voice Assistant** supporting regional Kannada dialects and 10+ Indian languages.
- 📈 **Real-time market prices** for 35+ commodities.
- 🌤️ **Weather forecasting** with farming-specific AI advice.
- 🏛️ **Government scheme information** with direct application filters.
- 💰 **Farm expense tracking** with category-wise analytics.
- 📞 **High Accessibility** through voice-first interactions.

## ✨ Features

### 🚜 ML-Powered Crop Recommendation
- Trained on **Kaggle Crop Recommendation Dataset**.
- **Random Forest Classifier** — 99.55% accuracy.

### 🐛 Pest & Disease Detection (Go Organic!)
- Upload plant leaf photo for instant AI analysis.
- **Go Organic** feature provides traditional/organic Indian wisdom (Neem, cow urine, etc.) for solutions.

### 🤖 Agentic AI Chatbot Advisor
- Powered by **Groq Llama 3.3 70B** for high-speed, intelligent reasoning.
- Supports regional Kannada dialects (Mangalorean, North Karnataka, Mysuru).
- Voice input & Auto-speak support for high accessibility.

### 💰 Financial Intelligence
- Track farm expenses, identify anomalies, and manage agricultural budgets.

## 🛠️ Installation & Setup

### Prerequisites
- Python 3.10+
- Git

### Step 1 — Clone & Setup
```bash
git clone https://github.com/INFOTHON-6-0/AGENTIC-25-SheCodes.git
cd AGENTIC-25-SheCodes
python -m venv venv
# Activate venv:
# Windows: venv\Scripts\activate
# Unix: source venv/bin/activate
pip install -r requirements.txt
```

### Step 2 — Run the Application
```bash
python app.py
```
Open browser: **http://127.0.0.1:7860**
